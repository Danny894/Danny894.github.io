Proyectos Personales

  • Dani Store - Dani Store es una tienda web dedicada a la venta de suéteres personalizados con logotipos de diferentes lenguajes de programación. Esta tienda está diseñada con HTML y CSS, ofreciendo una experiencia simple y visualmente atractiva para los entusiastas de la programación que quieren mostrar su lenguaje favorito en todas partes.
  • Carrito de compras - Este proyecto es un sitio web interactivo para visualizar y comprar cursos en línea. El diseño se centra en la presentación de los cursos, permitiendo al usuario agregar artículos a un carrito de compras, revisar su contenido y vaciarlo si es necesario. La página incluye un diseño atractivo y responsivo con diferentes secciones, como el encabezado, una barra de funciones, una lista de cursos y un pie de página con enlaces informativos.
  • Buscador de autos - Este proyecto es una sencilla aplicación web que permite a los usuarios buscar automóviles según diferentes criterios como marca, año, precio, número de puertas, transmisión y color.
  • Administrador de Pacientes de Veterinaria - Administrador de Pacientes de Veterinaria es una sencilla aplicación web diseñada para gestionar las citas de los pacientes veterinarios. Permite a los usuarios registrar datos básicos sobre las mascotas, sus dueños e información de la cita, como fecha, hora y síntomas.
